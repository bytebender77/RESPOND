Test with these curl commands. Backend must be running!

Quick Start

1. uvicorn api.main:app --reload
2. curl http://127.0.0.1:8000/setup
3. Open http://127.0.0.1:5500
POST/ingest/incident
curl -X POST "http://127.0.0.1:8000/ingest/incident" \
  -H "Content-Type: application/json" \
  -d '{"text": "Fire at Central Mall", "source_type": "social", "urgency": "critical"}'
POST/search/incidents
curl -X POST "http://127.0.0.1:8000/search/incidents" \
  -H "Content-Type: application/json" \
  -d '{"query": "fire emergency", "urgency": "critical", "limit": 10}'
POST/memory/incident/{id}/reinforce
curl -X POST "http://127.0.0.1:8000/memory/incident/{ID}/reinforce" \
  -H "Content-Type: application/json" \
  -d '{"source_type": "call", "text": "Caller confirms fire at mall"}'
PATCH/memory/incident/{id}/status
curl -X PATCH "http://127.0.0.1:8000/memory/incident/{ID}/status" \
  -H "Content-Type: application/json" -d '{"status": "acknowledged"}'
POST/recommend/actions
curl -X POST "http://127.0.0.1:8000/recommend/actions" \
  -H "Content-Type: application/json" -d '{"query": "fire emergency", "limit": 5}'
